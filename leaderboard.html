Evaluation Notes

  • All models evaluated in zero-shot setting (no few-shot examples, no chain-of-thought prompting) using the standardized CoTabBench prompt templates.
  • Overall score is the macro-average across all five task categories, weighted equally.
  • Scores for closed-source models reflect performance at a fixed API snapshot; scores may vary across model versions.
  • CoTabLLM is fine-tuned on CoTabInstruct (which is derived from training splits only); no test data leakage.
  • We welcome community submissions — see the GitHub repository for evaluation instructions and submission guidelines.
