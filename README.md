# CoTabBench — GitHub Pages Site

Official project website for **CoTabBench**, a real-world complex benchmark for Table QA with structural irregularity and content heterogeneity.

> Paper under anonymous review. Dataset and model weights will be released upon acceptance.

## Overview

This repository hosts the static website at [https://cotabbench.github.io](https://cotabbench.github.io), built with [Jekyll](https://jekyllrb.com/) and deployed via GitHub Pages.

The site includes:
- **Homepage** (`index.html`) — benchmark overview, statistics, method, and download section
- **Leaderboard** (`leaderboard.html`) — model evaluation results across task types and domains

## Local Development

**Prerequisites:** Ruby ≥ 3.1, Bundler

```bash
# Install dependencies
bundle install

# Serve locally with live reload
bundle exec jekyll serve --livereload
```

The site will be available at `http://localhost:4000`.

> The `_site/` directory is generated by Jekyll and is excluded from version control. GitHub Pages builds it automatically on push.

## Deployment

Push to the `main` branch. GitHub Pages will automatically build and deploy the site.

Make sure **GitHub Pages** is configured in repository Settings → Pages → Source: **Deploy from branch** → `main` / `(root)`.
